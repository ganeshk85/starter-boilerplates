To install

1. composer install

2. npm install

3. copy and paste .env file from github

4. edit .env file to add database host, port, databasename, username and password

5. create database called "cportal"

6. php artisan key:generate

7. php artisan migrate

8. php artisan db:seed

9. npm run dev

10. npm run watch
